• Circle
  • just another jQuery plugin

circleはちょっとしたメニュー用のjQueryプラグインです、左の図形をクリックしてその動きを確認してください

require : jquery 1.4+

ダウンロード：github , jsファイル

使い方：

html: 

<div id="circle">
  <a class="main"><img src="img/main.png" ></a>
  <a class="menu" href=""><img src="img/home.png" ></a>
  <a class="menu" href=""><img src="img/profile.png" ></a>
  <a class="menu" href=""><img src="img/schedule.png" ></a>
  <a class="menu" href=""><img src="img/group.png" ></a>
  <a class="menu" href=""><img src="img/add.png" ></a>
  <a class="menu" href=""><img src="img/search.png" ></a>
  <a class="menu" href=""><img src="img/config.png" ></a>
  <a class="menu" href=""><img src="img/add.png" ></a>
</div>

js: 

$(function(){
    $("#circle").circle();
});

もちろん，カスタマイズ用にパラメタは提供しています，例えば進行方向など。

以下のパラメタは提供しています、その値はデフォールト値です。

//メインボタンの直径
main : {
   width : 64
},
//初期位置、水平正方向を0度とする
dip : -15,
//サブボタン外側とメインボタン外側の距離
radius : 50,
//jquery easing
easing : "linear",
//各サブボタンのアニメーション時間
duration : 100,
//サブボタンの最終的に配置するエリアの範囲、初期位置から順時計に角度を計算する
range : 360,
//オープンアクションの終わりに発火する
open : function() {},
//クローズアクションの終わりに発火する
close : function() {},
//レイアウトレベル
zIndex : 0,
//延遅する倍率
delay : 1,
//画像を丸角にするかしないか
corner : true,
//方向(1 と -1だけが有効です)
direct : 1,
//クローズ動画をオープン動画の逆転にするかしないか
reveser : true,
//cursor
cursor : "pointer"

他に設定することもあるが、例えば動画の種類

左の図形をクリックしてみてください

animate : "panel"

これはwindowsのスタートパネルを真似る動きです。
